# 🛠️ Installation
### Windows
```
py -m pip install requests
```
### Linux
```
python3 -m pip install requests
```

# ⚔️ Usage
### Input your `hmac_signed_session` cookies inside `cookies.txt`, put your invite inside the tool, & done!
